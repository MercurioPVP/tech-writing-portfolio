# Портфолио технического писателя

Технический писатель с опытом разработки документации
по стандартам ЕСКД/ГОСТ. Специализация — понятные
инструкции, регламенты и пользовательская документация.

## Опыт

- Разработка конструкторской и эксплуатационной документации на производстве по ЕСКД/ГОСТ
- Нормоконтроль и приведение документов к единому стандарту

## Навыки

- Техническая документация: инструкции, руководства, регламенты
- Стандарты: ЕСКД, ГОСТ
- Инструменты: Markdown, Git, VS Code
