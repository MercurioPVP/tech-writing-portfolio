# Портфолио технического писателя

Технический писатель с опытом разработки документации
по стандартам ЕСКД/ЕСПД. Специализация — эксплуатационная документация, участие в испытаниях.
## Опыт

- Разработка, оформление и сопровождение конструкторской и эксплуатационной документации в соответствии с требованиями ЕСКД, ГОСТ и внутренними нормативами предприятия;
- Верстка и оформление документов в MS Word;
- Ведение и актуализация документации в PLM-системе;
- Актуализация шаблонов документации;
- Постановка и отслеживание задач по разработке документации в Jira, ведение и наполнение базы знаний;
- Сдача документации Заказчику;
- Оформление изменений и сопровождение извещений об изменении;
- Участие в испытаниях.

## Достижения

- Наладил взаимодействие отдела со службами предприятия, что ускорило выпуск документации.

## Навыки

- Техническая документация: инструкции, руководства, формуляры (паспорта), ведомости
- Стандарты: **ЕСПД**, **ЕСКД** 
- Инструменты: MS WORD, MS Visio, Jira, Confluence, Markdown, Git, VS Code
- Коммуникабельность, деловое общение, общение с разработчиками (поиск информации)
