# Как импортировать события в Google Календарь из файла с расширением .ics

Это руководство показывает, как добавить события из файла с расширением .ics (iCalendar) в Google Календарь. Подходит для готовых расписаний, приглашений и подписок на события.

## Перед началом

Убедитесь что у вас есть:
- аккаунт Google;
- файл с расширением .ics;
- доступ к веб-версии Google Календарь.

> Импорт через мобильное приложение недоступен. На телефоне откройте Google Календарь через любой браузер в режиме "Версия для ПК".

## Импорт файла

1. Откройте Google Календарь в браузере.
2. Нажмите "шестеренку" в верхней правой части страницы.
3. Выберите пункт "Настройки" в выпавшем списке.
4. Выберите раздел "Импорт и экспорт" на панели в левой части страницы.
5. Нажмите на пространство "Выберите файл на компьютере".
6. Найдите файл с расписанием в расширение .ics на своем устройстве.
7. Выберите один из своих календарей в пространстве "Добавить календарь".
>Важно: cобытия добавятся именно в выбранный календарь — если выбрать не тот, потом придётся удалять вручную.
8. Нажмите на кнопку "Импорт".
9. Дождитесь появления окна вида "Мероприятий импортировано: X из Y", где X - это количество успешно импортированных мероприятий, а Y общее количество мероприятий в файле.

## Как проверить результат
1. Вернитесь на основную страницу Google Календарь.
> Важно: необходимо представлять примерную дату (хотя бы месяц), на которую будет добавлено мероприятие.
2. Откройте Google Календарь в формате "Месяц", для этого:
     - нажмите на область "Неделя", которая располагается справа от найденной ранее "шестеренки".
     - выберите "Месяц" в выпавшем списке.
3. Найдите месяц куда должно было добавиться мероприятие с помощью стрелок в верхней левой части страницы.
4. Проверьте параметры и временные рамки установленных мероприятий и, при необходимости, откорректируйте их под себя.

## Возможные проблемы
| Проблема | Причина | Решение |
|----------|---------|---------|
| После нажатия кнопки "Импорт" появлется окно "Мероприятий импортировано: 0 из 0. Не удается обработать файл iCal/CSV."   | Выбран файл с расширением отличным от .ics  | Выбрать файл с расширением .ics |
| Не получается найти мероприятия в Google Календарь   | Выбран неверный календарь    | Проверить все календари на наличие добавленных мероприятий  |
|
