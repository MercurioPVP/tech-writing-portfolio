# Горячие клавиши VS Code (Windows / Linux)

Справочник содержит основные сочетания клавиш редактора VS Code. Клавиши указаны для Windows и Linux. В macOS вместо клавиши "Ctrl" используется "Cmd".

## Навигация

| Действие | Клавиши |
|----------|---------|
| Открыть палитру команд | `Ctrl+Shift+P` |
| Переход к файлу | `Ctrl+P` |
| Перейти к строке | `Ctrl+G` |
| Перейти к определению | `F12`|

## Редактирование

| Действие | Клавиши |
|----------|---------|
| Сместить табуляцию строки | `Shift+Tab` |
| Закомментировать строку | `Ctrl+/`|
| Удалить строку целиком | `Ctrl+Shift+K`|
| Переименовать символ | `F2` |
| Выделить все вхождения слова | `Ctrl+Shift+L` |

## Поиск и замена

| Действие | Клавиши |
|----------|---------|
| Заменить текст в файле | `Ctrl+H` |
| Заменить текст во всем проекте | `Ctrl+Shift+H` |
| Искать в файле | `Ctrl+F` |
| Искать по проекту | `Ctrl+Shift+F` |
