# Горячие клавиши VS Code (Windows / Linux)

Справочник содержит основные сочетания клавиш редактора VS Code. Клавиши указаны для Windows и Linux. В maсOS вместо клавищи "Ctrl" используется "Cmd".

## Навигация

| Действие | Клавиши |
|----------|---------|
| Открыть палитру команд | `Ctrl+Shift+P` |
| Быстрый переход к файлу | `Ctrl+P` |
| Перейти к строке | `Ctrl+G` |
| Перейти к переменной | `Alt+ПКМ`|

## Редактирование

| Действие | Клавиши |
|----------|---------|
| Сместить табуляцию строки | `Shift+Tab` |
| Закомментировать строку | `Ctrl+/`|
| Удалить строку целиком | `Shift+Del`|

## Поиск и замена

| Действие | Клавиши |
|----------|---------|
| Переименовать переменную втнутри блока | `F2` |
| Выделить все вхождения слова | `Ctrl+Shift+L` |
| Замена текста во всем проекте | `Ctrl+Shift+H` |
| Перейти к конкретной строке | `Ctrl+G` |
